# menyew releases

Public update feed for [menyew](https://github.com/ConceptHouse/menyew) — a macOS menu-bar monitor.

- **`appcast.xml`** is the [Sparkle](https://sparkle-project.org) update feed the app checks.
- Notarized builds are attached as **release assets** under Releases.

The application source lives in the private `ConceptHouse/menyew` repository.
